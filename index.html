(الجزء الأول) لوحة تحكم المدرس

التحكم الكامل في إنشاء وإدارة الكويزات

1. إدارة وإنشاء الكويزات

  • ✓ البيانات الأساسية (عنوان، سعر، مدة).
  • ✓ إعدادات متقدمة (خلط الأسئلة والإجابات).
  • ✓ التحكم في إظهار النتيجة (فوري أو مؤجل).

2. بنك الأسئلة

  • ✓ يدعم: نص، صورة (للمعادلات)، فيديو.
  • ✓ تحديد الخيارات (MCQ) والدرجات.
  • ✓ ميزة "بنك الأسئلة" لإعادة الاستخدام.

3. إدارة الطلاب والمرونة

  • ✓ ميزة "الإضافة اليدوية (المجانية)".
  • ✓ إرسال بيانات الدخول للطالب تلقائياً.
  • ✓ "عداد" لمتابعة الإضافات المجانية.

4. متابعة النتائج والتحليلات

  • ✓ الاطلاع على درجات كل طالب.
  • ✓ معرفة "متوسط الدرجات" للكويز.
  • ✓ تحديد "أصعب سؤال" و "أسهل سؤال".

5. متابعة الأرباح والنشر

  • ✓ لوحة متابعة مالية بسيطة للمبيعات.
  • ✓ تقرير أرباح وعدد مشترين لكل كويز.
  • ✓ الحصول على "رابط خاص" لنشر الكويز.

(الجزء الثاني) رحلة الطالب

من الضغط على الرابط إلى الحصول على النتيجة

خطوة 1: الاكتشاف والدفع

يضغط الطالب على الرابط، تظهر له صفحة الكويز (السعر والوصف). ثم ينتقل للدفع الآمن (مدى، Apple Pay).

خطوة 2: الحصول على الدخول

بعد الدفع، يقوم النظام **تلقائياً** بإنشاء حساب للطالب وإرسال **إيميل بكلمة المرور**.

خطوة 3: تجربة الكويز والنتيجة

يسجل الطالب الدخول ليجد مكتبته، يبدأ الكويز مع **عداد وقت**، وعند الانتهاء **تظهر النتيجة فوراً**.
